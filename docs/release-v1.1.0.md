# TinyPM v1.1.0 Release Notes

TinyPM v1.1.0 introduces the AI-native project management loop.

## Highlights

- AI assistant workspace.
- Project data MCP tools.
- Project review report.
- Single plan review.
- Risk list generation.
- Weekly project summary.
- Project plan Excel import/export.

## Suggested GitHub Release Title

```text
TinyPM v1.1.0 - AI-native project management for small teams
```

## Suggested GitHub Release Body

```markdown
TinyPM v1.1.0 turns the project management data model into an AI-readable workspace.

This release adds an AI assistant workspace, project data MCP tools, Excel import/export for project plans, project review reports, single plan review, risk list generation, and weekly project summaries.

Use TinyPM when you want a self-hosted project management system where an AI assistant can work with real project data instead of vague chat context.
```
