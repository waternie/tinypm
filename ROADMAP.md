# TinyPM Roadmap

## Current Focus

TinyPM is focused on lightweight self-hosted project management with AI-assisted project review.

## Near Term

- Improve project data MCP tools.
- Add more project management agent skills.
- Expand Excel import/export coverage.
- Add project-level audit logs.
- Improve weekly summary and risk report prompts.

## Mid Term

- More granular permission control.
- File uploads and project document management.
- Cost analytics and reporting.
- More export formats for project review and weekly reports.
- Better onboarding demo data.

## Later

- Multi-team workspace support.
- More model providers.
- Role-specific project assistants.
- Integration with external tools such as GitHub, GitLab, Jira, Feishu, or Slack.

## Non-goals

- TinyPM is not trying to become a full enterprise PM suite.
- TinyPM is not a general AI chat platform.
- TinyPM does not replace project owners or human review.
