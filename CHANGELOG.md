# Changelog

## v1.1.0

TinyPM v1.1.0 adds the AI assistant and project data workflow needed to position TinyPM as an AI-native project management platform.

### Added

- AI assistant workspace.
- DeepSeek-compatible model settings.
- Project-scoped agent sessions.
- Project data MCP tools.
- Project review report.
- Single plan review.
- Risk list generation.
- Weekly project summary.
- Project plan Excel import/export.

### Improved

- README positioning for self-hosted AI-native project management.
- Docker Compose quick start documentation.
- Demo and architecture documentation.

## v1.0.0

Initial public version with:

- User authentication and roles.
- Member library.
- Project management.
- Milestones, plans, requirements, and issues.
- Project announcements.
- Cost management.
- Docker Compose deployment.
